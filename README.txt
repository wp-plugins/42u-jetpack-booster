== Description ==

The 42U Jetpack Booster adds redirect tags and HTML email templates to Jetpack Contact Forms.

To add a redirect to your Jetpack Contact Form, just add the following tag to the HTML view of your Jetpack Contact Form:

[contact-field label="redirect" type="hidden" default="/thanks/" /]

Set the default value to the location where you would like the form to redirect after submission.
        
== Installation ==

1. Upload the `42u-jetpack-booster` directory (including all files within) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
